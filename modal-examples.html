Système de Modal Universel

Démonstration des différentes utilisations du système de modal réutilisable

Modal Simple

Affiche un message simple avec bouton OK

universalModal.alert({ title: 'Information', message: 'Ceci est un message d\'information simple.', icon: 'fa-info-circle' });

Modal Confirmation

Demande une confirmation avant une action

universalModal.confirm({ title: 'Confirmation', message: 'Êtes-vous sûr de vouloir supprimer cet élément ?', icon: 'fa-exclamation-triangle', onConfirm: function() { console.log('Action confirmée'); } });

Modal Succès

Affiche un message de succès

universalModal.success({ title: 'Succès !', message: 'L\'opération a été effectuée avec succès.', icon: 'fa-check-circle' });

Modal Erreur

Affiche un message d'erreur

universalModal.error({ title: 'Erreur', message: 'Une erreur est survenue. Veuillez réessayer.', icon: 'fa-exclamation-circle' });

Modal Formulaire

Affiche un formulaire personnalisé

universalModal.form({ title: 'Ajouter un utilisateur', icon: 'fa-user-plus', content: '... formulaire HTML ...', onSubmit: function() { console.log('Formulaire soumis'); } });

Modal avec Onglets

Affiche un modal avec plusieurs onglets

universalModal.withTabs({ title: 'Paramètres avancés', tabs: ['Général', 'Sécurité', 'Notifications'], content: '... contenu avec onglets ...' });

Modal Chargement

Affiche un indicateur de chargement

const modal = universalModal.loading({ title: 'Traitement en cours', message: 'Veuillez patienter...' }); // Fermer après traitement setTimeout(() => universalModal.close(modal), 3000);

Modal Personnalisé

Modal entièrement personnalisé

universalModal.open({ title: 'Modal personnalisé', size: 'large', type: 'success', content: '... contenu HTML personnalisé ...', buttons: [ { text: 'Annuler', type: 'secondary' }, { text: 'Valider', type: 'primary' } ] });

Documentation API

Référence complète des méthodes et options disponibles

Méthodes principales

// Ouvrir un modal universalModal.open(options) // Fermer un modal universalModal.close(modal) // Fermer tous les modaux universalModal.closeAll()

Types prédéfinis

universalModal.alert(options) universalModal.confirm(options) universalModal.success(options) universalModal.error(options) universalModal.loading(options) universalModal.form(options) universalModal.withTabs(options)

Options principales

{ title: 'Titre', subtitle: 'Sous-titre', icon: 'fa-icon', type: 'default|success|error|warning|info', size: 'small|medium|large|fullscreen', content: 'HTML content', message: 'Text message', buttons: [...], onOpen: function() {}, onClose: function() {}, onConfirm: function() {} }

Configuration des boutons

buttons: [ { text: 'Texte du bouton', type: 'primary|secondary|success|danger|outline', icon: 'fa-icon', onclick: 'javascript', disabled: false } ]
